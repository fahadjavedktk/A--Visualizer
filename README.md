# A* Pathfinding Visualizer

An interactive visualization of the A* search algorithm using Python.

## Features
- Grid-based pathfinding
- Real-time node exploration
- Mouse interaction (draw obstacles)
- Shortest path visualization

## Controls
- Left Click: Add walls
- Right Click: Remove walls
- SPACE: Start algorithm

## Tech Stack
- Python
- Pygame

## How to Run
pip install -r requirements.txt
python shortestpathvisualizer.py
